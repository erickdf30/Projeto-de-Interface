# ⚡ TechNova — Projeto Web HTML/CSS

Site multípáginas desenvolvido com HTML5 e CSS3 sobre o tema **Tecnologia & Inovação**.

## 📁 Estrutura de Arquivos

```
projeto-web/
├── index.html          ← Página principal
├── galeria.html        ← Galeria com 5 imagens
├── tabelas-midia.html  ← Tabelas + Vídeo + Áudio
├── formulario.html     ← Formulário completo
├── style.css           ← Estilos CSS (10+ parâmetros)
└── README.md           ← Este arquivo
```

## ✅ Requisitos Atendidos

### index.html
- [x] 2 títulos centralizados com hierarquia (H1 e H2)
- [x] 4 parágrafos com mínimo de 4 linhas cada
- [x] 10 palavras/frases com negrito, sublinhado e itálico
- [x] 3 listas diferentes (ul, ol, dl) com 6 itens cada e separação visual
- [x] Mínimo 2 links para outras páginas HTML
- [x] Favicon com ícone ⚡ e title na aba
- [x] Menu com `<nav>` linkando para 2+ páginas dentro de `<header>`
- [x] Imagem de entrada abaixo do menu/header
- [x] Âncora interna (#topo, #sobre, #listas)

### galeria.html
- [x] 5 imagens com mínimo 2 parágrafos de texto

### tabelas-midia.html
- [x] 2 tabelas com variações (estilos, badges coloridos, colspan)
- [x] 1 vídeo com `<video controls>`
- [x] 1 áudio com `<audio controls>`

### formulario.html
- [x] 10+ opções diferentes com `<form>` e `<fieldset>`
- [x] Radio buttons (gênero, formato, etc.)
- [x] Checkboxes (interesses, comunicações)
- [x] Botão reset (`type="reset"`)
- [x] Botão enviar (`type="submit"`)

### Todas as páginas
- [x] Âncoras de forma livre em todas as páginas
- [x] Arquivo `style.css` com 10+ parâmetros de estilização

### style.css — Parâmetros CSS
1. `font-family` — Tipografia customizada (Google Fonts)
2. `background-color` / `background` — Gradientes e cores
3. `color` — Cores de texto
4. `border` / `border-radius` — Bordas e arredondamento
5. `padding` / `margin` — Espaçamento
6. `display: grid` / `flex` — Layout moderno
7. `transition` — Animações suaves
8. `box-shadow` — Sombras e efeitos
9. `font-size` / `font-weight` — Tamanho e peso da fonte
10. `hover` states — Interatividade
11. CSS Variables (`--var`) — Sistema de design
12. `@media queries` — Responsividade mobile

## 🚀 Como publicar no GitHub Pages

1. Crie um repositório no GitHub
2. Faça upload de todos os arquivos
3. Vá em **Settings → Pages**
4. Selecione a branch `main` e pasta `/root`
5. Clique em **Save**
6. Acesse: `https://SEU_USUARIO.github.io/NOME_DO_REPO/`
